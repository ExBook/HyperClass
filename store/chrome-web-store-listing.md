```
HyperClass 是一个为中小学老师打造的「课堂工具箱」。点击浏览器工具栏图标，就能在新标签页打开一组干净、好看、可大屏演示的课堂互动工具 —— 老师讲解，学生上台动手。

🎯 为什么适合课堂

· 演示 + 练习：很多工具既能老师演示讲解，又能出题让学生练习并自动批改；练习态不泄露答案颜色，杜绝看色作弊。
· 学科严谨：地理与数学在细节上经得起推敲（直射点、晨昏线、极昼极夜、等高线判读…），可直接当板书。
· 离线运行：工具内置，无需联网即可上课。

🧰 内置 16 个工具
通用（8）：随机点名、随机分组、小组计分板（实时排行榜，最多 64 组）、抽题盒、计时器、幸运转盘、骰子、噪音检测仪。
地理（7）：中国省级行政区、经纬网定位、中国气候类型、气候直方图判读、中国四大地理区域、等高线地形图、地球运动演示（公转 + 自转）。
数学（1）：函数图像（一次 / 二次函数实时绘制）。

🔒 隐私优先
完全在本地运行，不收集任何个人信息，不上传数据，无广告、无统计。麦克风仅「噪音检测仪」在你授权后用于实时显示音量，不录音、不保存。

💌 没有你想要的工具？
把需求（年级 / 学科 / 想要的演示或练习方式）发给我们：hyperclass@163.com

开源（MIT），同时提供桌面版与网页版：https://github.com/ExBook/HyperClass
```

## ④ 隐私权规范 Privacy practices（⚠️ 必须与扩展实际行为一致）

> 本扩展实测：**无任何声明权限**、**无远程代码**、**无网络请求**、**无统计/广告**。下面内容据此填写，审核风险最低。

**单一用途说明 Single purpose**（粘贴）
```
HyperClass 是一个面向中小学课堂的教学工具箱。点击扩展图标后，在新标签页打开一组供教师课堂使用的互动小工具（随机点名、计时、计分、地理与函数的演示与练习等）。所有功能均在本地运行。
```

**权限说明 Permission justifications**
```
本扩展未在 manifest 中申请任何权限（无 permissions、无 host_permissions、无 content_scripts）。
点击工具栏图标仅通过 chrome.action 触发，在新标签页打开内置页面（chrome.tabs.create 创建新标签页无需 "tabs" 权限）。
因此本项无需逐条填写权限理由。
```
> 后台若因 manifest 没声明权限而**不显示**此区块，属正常——直接跳过即可。

**「噪音检测仪」用到麦克风的说明**（如审核问及，可用此话术）
```
仅「噪音检测仪」工具使用麦克风：在用户点击允许浏览器的标准授权弹窗后，实时读取环境音量用于显示分贝表，不录音、不保存、不上传，关闭工具即停止。该能力通过网页运行时的 getUserMedia 获取，并非扩展声明的权限。
```

**是否使用远程代码 Are you using remote code?**
```
否（No）。所有 HTML / JS / CSS / 资源均打包在扩展内，无远程脚本、无 eval、无外部加载。
```

**数据用途 Data usage / 收集的用户数据类别**
```

说明：偏好（名单 / 主题 / 计分等）仅以 localStorage 保存在用户本地浏览器，不属于"收集"，
```

为什么适合课堂

少一点折腾，多一点上课的节奏感

每个工具都围绕真实课堂动作设计：投屏清楚、反馈明显、练习态不露答案，能快速从讲解切到学生操作。

01

投屏清楚

大字号、高对比、稳定布局，后排学生也能看清题目、地图、计分变化和结果反馈。

02

能讲也能练

地理类工具支持演示和练习，学生拖放或选择后可批改，练习时不靠颜色泄露答案。

03

离线运行

桌面版安装后无需联网，适合多媒体讲台、临时借用电脑和网络不稳定的教室。

04

三端同源

同一套工具可作为桌面软件、网页和 MV3 浏览器扩展使用，后续迭代维护成本更低。

网页和扩展

也可以免安装使用

如果学校电脑不能安装软件，可以把 Web 版部署成静态站；浏览器扩展则适合固定在工具栏快速打开。

网页

项目主应用执行 `npm run build` 后部署 `dist/`。产品主页部署时直接上传 `website/`。

浏览器扩展

执行 `npm run build:ext` 生成 `dist-ext/`，在 Chromium 浏览器中加载已解压扩展。

源码

代码已开源，欢迎按课堂需求继续添加工具、截图和部署说明。
